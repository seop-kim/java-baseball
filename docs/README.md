## 숫자 야구 주요 기능
- [ ] 컴퓨터가 1~9까지 서로 다른 랜덤 수 3개를 만드는 기능
- [ ] 플레이어가 1~9까지 서로 다른 수 3 개를 입력하는 기능
- [X] 스트라이크(같은 자리에 같은 값) count 기능
- [X] 볼(다른 자리에 같은 값) count 기능
- [ ] 낫싱(같은 값이 하나도 없을 경우) 기능


## 예외 처리
- [ ] 플레이어가 값을 잘 못 입력할 경우 예외 발생


## 입출력
### 입력
- [ ] 숫자 입력지
- [ ] 재시작 여부 입력

### 출력
- [ ] 게임 시작 메시지 출력
- [ ] 숫자 입력 메시지 출력
- [ ] 게임 현황 메시지 출력
- [ ] 게임 종료 메시지 출력
- [ ] 게임 재시작 여부 메시지 출력
